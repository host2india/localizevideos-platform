# LocalizeVideos Platform

Desktop-first AI-powered video localization platform.

## What this is
LocalizeVideos allows creators, educators, studios, and businesses to:
- Transcribe videos
- Translate to any language
- Generate AI voiceovers
- Create subtitles
- Export localized videos

Supports both:
- Desktop software
- SaaS backend (credits + billing)
